## 2. [많이 나오는 알고리즘 유형](#목차)
### 2-1. [2 차원 배열]()
#### 2-1-1. [방향배열]()
```
'2차원 배열' 에서 특정 점을 기준으로 어떤 일이 일어난다면 고려할 수 있는 알고리즘.

예를 들어, 2차원 배열에서 + 모양이나 X 모양으로 어떤 사건이 일어난다면 사용 가능하다.

주의 사항 : '범위'
특정 점을 기준으로 뻗어나가기 때문에 2차원 배열의 범위를 벗어나게 되는 경우가 있다.
조건문 (if) 를 사용하여 범위를 지정해 주도록 하자.
```
